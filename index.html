OVI DEL CURSO

INTRODUCCION

Esta ovi esta orientada al desarrollo web donde se evidenciarán documentos, videos y enlaces relacionados con los temas de Introducción, profundización e información complementaria sobre el uso de GIT y GITHUB para gestionar proyectos colaborativos de diseño web, HTML5 y CSS3 para creación de contenidos en formato web (Paginas y Sitios Web). Con la finalidad de conocer mas sobre el tema y lograr un correcto manejo de estas herramientas para el desarrollo web.

OBJETIVO GENERAL

Crear un ovi relacionada con el curso de Diseños de sitios web con la finalidad de asociar contenido de interés sobre el tema de GIT GITHUB y manejo de html5 y css3 en el desarrollo de sitios web.

¿QUE ES DESARROLLO WEB?

El desarrollo web se enfoca en la planificación de un sitio web destinado a un usuario final, definiendo los objetivos que se deben tener presente para su desarrollo. Definiendo el formato y el lenguaje de programación apropiado para este proyecto, si es el caso también de desarrolla el diseño de la base de datos. Todo esto con el fin de crear soluciones tecnológicas que van a ser implementadas dentro de un navegador..

HTML5 Y CSS3

Hoy en día todo diseñador gráfico que se oriente hacia la web debe conocer los lenguajes de marcado y de estilo HTML y CSS. Conocer dichos lenguajes les permite transformar diseños en experiencias interactivas, fundamental para un buen desarrollo web. El HTML en su versión 5 nos permite maquetar sitios web a través de etiquetas y dichas etiquetas las podemos convertir en objetos visuales gracias al CSS en su versión 3.
